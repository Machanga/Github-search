Github Search

This is an angular app that a user can use to search for any Github User through the username and view the following details in a click of a button:

  1. Username
  2. login name
  3. Email address
  4. Bio
  5. Number of followers
  6. Number of people they are following
  7. Number of repositories
  8. Account creation date.
  1. Repository name
  2. Repository description
  3. Repository url
